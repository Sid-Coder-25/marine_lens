# FinSight

FinSight is a mobile-first fish identification and conservation guidance prototype built with only HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` - app structure and accessible UI.
- `style.css` - responsive ocean-themed styling.
- `script.js` - upload, validation, preview, and mock classification behavior.
- `fish-data.js` - local mock species and conservation records.
- `assets/ocean-lines.svg` - subtle local background texture.

## How to Use

Open `index.html` in a browser.

1. Drag and drop a fish image, choose a local photo, or use the camera button on a smartphone.
2. The app validates JPG, JPEG, PNG, and WEBP files up to 10 MB.
3. Preview the selected image and press `Identify Fish`.
4. Review the simulated species match, conservation status, guidance, and alternative mock matches.
